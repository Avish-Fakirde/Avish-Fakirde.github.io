🎵 Feature-Based Speech-Music Classification

Developed a comprehensive system for automatic speech-music classification using multiple acoustic features and machine learning techniques. Evaluated 17 different acoustic features including chroma-based, spectral, and temporal features. Compared traditional Gaussian Mixture Models (GMM) with Convolutional Neural Networks (CNN) on GTZAN and LEAP Lab datasets, with Chroma High-Frequency Energy (CHFE) feature showing particularly strong performance.

Machine Learning Speech Processing CNN GMM

🎨 AnimeGen: Generating Anime-Style Images Using DCGAN and Style Transfer

Combined Deep Convolutional Generative Adversarial Networks (DCGAN) and neural style transfer to generate anime-style images from human faces. The DCGAN generates realistic face images, which are then transformed using style transfer with anime images as style references. Demonstrates how generative models can be enhanced with artistic style transfer techniques using VGG19 architecture.

DCGAN Style Transfer VGG19 Deep Learning

🖼️ Poisson Image Matting

Studied and implemented techniques for global and local Poisson matting to extract foreground objects from natural images. Used the Gauss-Seidel method to solve the Poisson equation for reconstructing smooth alpha mattes. Applied guided filtering and interactive gradient manipulation to refine results for complex images, ensuring accurate foreground-background separation.

Python Image Processing Computer Vision Numerical Methods
